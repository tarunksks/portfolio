# Tarun K.S — Portfolio

Simple, fast, single-page personal portfolio built with **HTML + CSS + JavaScript**.

## Run locally

You can open `index.html` directly in your browser, or use a small static server.

### Option A: VS Code Live Server
- Install the **Live Server** extension
- Right-click `index.html` → **Open with Live Server**

### Option B: Python

```bash
python -m http.server 5500
```

Then open `http://localhost:5500`.

## Deploy

### GitHub Pages
1. Push this folder to a GitHub repo
2. In GitHub: **Settings → Pages**
3. Set:
   - Source: **Deploy from a branch**
   - Branch: `main` / root
4. Save — your site will be published

### Netlify (drag & drop)
1. Go to Netlify
2. Drag & drop the folder containing `index.html`

## Customize
- Update text/links in `index.html`
- Colors and layout in `styles.css`
